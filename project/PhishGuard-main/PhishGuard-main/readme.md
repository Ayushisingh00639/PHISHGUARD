\## Description

PhishGuard is a beginner-friendly \*\*Phishing Email Detector\*\* built with \*\*Streamlit\*\*.  

It checks emails for suspicious words, links, and sender patterns to help identify phishing attempts.



---



\## Features

\- Detects suspicious keywords like: `urgent`, `verify`, `login`, `password`, etc.  

\- Checks for suspicious URLs (`.xyz`, `.top`, `.io`, `login`, `secure`)  

\- Analyzes sender email for generic addresses (`no-reply`, `support@`, etc.)  

\- Color-coded results:  

&nbsp; - 🟢 Legitimate  

&nbsp; - 🟠 Potential Phishing  

&nbsp; - 🔴 High Risk Phishing  

\- Expandable section showing \*\*why an email is flagged\*\*  

\- Gradient progress bar \& hacker-style neon design
